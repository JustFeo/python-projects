# Weather App

This is a simple command-line tool that retrieves current weather information for a specified location. It uses the [wttr.in](https://wttr.in) service which does not require an API key.

## Requirements

- Python 3.7+
- `requests` library

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Usage

```bash
python weather.py <location>
```

Example:

```bash
python weather.py London
```

This will output a brief weather summary for London.
